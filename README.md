# CodeAlliance

## Summary
 CodeAlliance is an online marketplace for software and datasets produced at universities.


## Technologies

### Tools

  * Atom
  * Sketch

### Backend
  * Ruby 2.4.0
  * Rails 5.0.2
  * Heroku
  * PostgreSQL
  * Devise

### Front-End
  * HTML5
  * Sass
  * JavaScript
  * jQuery
